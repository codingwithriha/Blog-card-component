* Styled using Tailwind CSS

## 📂 Project Setup

Clone the repository:

```bash
git clone https://github.com/codingwithriha/Blog-card-component
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
